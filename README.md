# CTFNmapScanner
An easy way to manage nmap scans during a CTF

# TODO

# Usage:
|Command Name/Usage |  Use  |
|-------------------|-------|
|scan [ip [ip2...]] | Scan IPs |
|     show          | Show hosts scanned |
| list [ip]         | List ports for the specified IP |
| help              | Lists commands that can be used |
